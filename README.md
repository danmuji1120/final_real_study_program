# 설명

---

영어단어, 한자, 수도 등의 각종 문제들을 추가하고 시험을 볼 수 있는 프로그램

# 실행

터미널에 입력

```
npm i
```

실행

```
npm start
```

# 사용방법

---

### 1. 시험과목 추가

### 2. 과목 안에 챕터 추가

### 3. 시험 보기

## #추가 기능

리마인드 시스템

- 일정 시간 시험을 보지 않았거나 점수가 낮은 문제들을 다시 시험볼 수 이도록 추천해준다.

오답노트 시스템

- 시험이 끝난 후 틀린 문제들을 다시 볼 수 있도록 틀린 문제들로만 다시 시험을 본다.

# 추가 예정 기능

하드모드

- 오답률이 높은 문제들로만 시험문제를 재구성한다.
